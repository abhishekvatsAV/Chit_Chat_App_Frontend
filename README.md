## To Run Locally

First Put backend and frontend folder inside a new folder

then first open terminal in both backend and frontend folder.  
And Run these cmnd

### backend
1. npm install
2. npm start

### frontend
1. npm install
2. npm start

Now app will run on [http://localhost:3000/].
